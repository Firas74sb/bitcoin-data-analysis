Analyse du Réseau Bitcoin

Ce outil de visualisation interactif vous permet d'explorer les réseaux de transactions Bitcoin à différents points temporels. Vous pouvez analyser la structure du réseau, identifier les participants clés et suivre les flux de transactions. L'outil propose deux modes principaux :

  • Analyse à un Moment Donné : Examinez la structure du réseau, les communautés et les nœuds clés pour une heure spécifique
  • Analyse Comparative : Comparez les états du réseau entre deux points temporels différents pour identifier les changements

Les fonctionnalités incluent la détection de communautés, l'identification des principaux dépensiers/récepteurs, l'analyse de la valeur des transactions et l'exploration interactive du réseau.

Sélectionnez un mode d'analyse pour commencer :
