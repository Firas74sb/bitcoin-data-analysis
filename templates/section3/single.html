Single Time Analysis

This tool allows you to analyze Bitcoin transaction networks at specific time points. You can explore the network structure, identify key participants, and examine transaction patterns for any selected hour.

Key features include:

  • Interactive Network Visualization: Explore the transaction network with an interactive force-directed graph where node size reflects transaction volume and edge thickness shows value.
  • Detailed Statistics: View comprehensive metrics including total nodes, transactions, and value flows with both satoshi and USD equivalents.
  • Top Participants: Identify the most active nodes in the network - the top spenders, receivers, and those that are both.
  • Community Detection: Automatically detect and analyze transaction communities to understand how the network clusters.
  • Advanced Search: Search for specific nodes or communities and examine their transaction patterns in detail.

To begin, select a date and hour from the dropdown menus below. The visualization will automatically update to show the top transactions during that time period.
